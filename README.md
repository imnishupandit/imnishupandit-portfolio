### 2. HR Analytics Dashboard ‚Äî Employee Attrition Analysis
A business-focused HR analytics project aimed at understanding employee
attrition patterns and key factors contributing to employee exits.

**What I worked on:**
- Cleaned and prepared HR data using Excel
- Handled missing values and inconsistent records
- Created analysis-ready datasets
- Designed and built an interactive Power BI dashboard

**Key Insight:**  
Attrition is highest among early-career employees in lower salary bands,

Research & Development and Sales roles.
